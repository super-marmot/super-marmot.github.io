# super-marmot.github.io

Landing page for [Marmot](https://github.com/stancsz/marmot) — on-device AI chat + agents for iOS and Android.

**Live:** https://super-marmot.github.io

The page is authored in the app repo at `stancsz/marmot` → `docs/index.html`
(single file, no build step). Edit there, then sync here:

```bash
cp ../marmot/docs/index.html index.html
cp ../marmot/docs/assets/*.svg assets/
```
